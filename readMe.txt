Voordat je de server start zorg ervoor dat node_modules aanwezig is.
mocht dit niet het geval zijn voer dan het volgende commando in het opdrachtvenster: npm install

Om de server te starten:

Lshift+Lalt+ rechter muis knop
in de huidige map

klik op:"Opdrachtvenster hier openen".

voer in het opdrachtvester het volgende commando in: node localServer.js 
druk op enter

Open Main Menu: "public/MainLevelSelect.html"
